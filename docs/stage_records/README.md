# 阶段文档目录说明

本目录用于统一存放项目各阶段固定生成的：

- 目标/计划文档
- 交接文档
- 评测说明文档
- 版本基线文档

命名规则如下：

- `stage_01_*`：第一阶段文档
- `stage_02_*`：第二阶段文档
- `stage_03_*`：第三阶段文档
- `stage_04_*`：v1.0 基线阶段文档

当前文件说明：

- `stage_01_project_handoff_summary_v1.md`
  - 第一阶段总体交接摘要
- `stage_02_project_handoff_summary_v2.md`
  - 第二阶段更细的暂停点与收口交接说明
- `stage_03_next_phase_deepening_plan_v1.md`
  - v1.0 之前形成的下一层级深化计划文档
- `stage_04_v1_0_runnable_baseline_and_next_plan.md`
  - 当前第一个可跑通版本 v1.0 的基线与下一层级详细计划
- `stage_04_v1_0_evaluation_handoff.md`
  - 当前 v1.0 的评测说明与实现程度判断

后续新增阶段文档时，继续沿用：

- `stage_05_*`
- `stage_06_*`

这样的编号方式，保证一眼可以看出“属于哪个阶段、是什么类型的文档”。
