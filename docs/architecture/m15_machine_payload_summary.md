# M15 机器层摘要卡增强

## 目标

在 `M13` 完成结果页分区导航、`M14` 完成脚本层快速导航之后，继续提升结果页对结构化信息的扫描效率，优先优化当前仍以原始 JSON 为主的机器层展示。

本阶段目标：
- 为 `machine_payload_layer` 增加一组摘要卡，先展示顶层关键信息，再保留完整 JSON 原文。
- 不改动后端返回结构，不新增任何派生接口。
- 保持“先摘要，后真值”的展示策略，避免掩盖原始数据。

## 原材料来源

本阶段直接参考：
- `/home/admin2/smy/upstream-materials/react-bits-main/src/ts-default/Components/BorderGlow/BorderGlow.tsx`

## 适配原则

- 保留上游 `BorderGlow` 的核心语义：用更强的边界感强调高价值信息卡。
- 不引入上游的指针发光逻辑与复杂渐变变量系统，只借“强调型信息卡”的视觉语义。
- 摘要卡只展示现有 JSON 的顶层键、值类型和简要内容，不生成新的业务字段。
- 完整 JSON 原文继续保留在原位置，摘要卡不能替代真值。

## 变更范围

### 机器层
- 在机器层 JSON 原文前新增摘要卡区域。
- 每张摘要卡至少包含：键名、值类型、简短预览。
- 对对象、数组、字符串采用不同的轻量预览策略。

### 结果页样式
- 新增摘要卡网格、标签、预览样式。
- 不改动多模态层、平台层和脚本层结构。

## 当前阶段明确不做

- 不改动后端 schema。
- 不隐藏或替换原始 JSON。
- 不引入新的动画依赖。

## 验证要求

- 变更后执行 `npm run build`。
- 同步更新 `docs/changelog.md`。
- 以同一批次 Git 提交收口。
