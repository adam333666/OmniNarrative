# M16 多模态层摘要卡增强

## 目标

在 `M15` 完成机器层摘要卡增强之后，继续统一结果页对原始结构块的展示策略，优先优化仍以整块 JSON 展示的多模态层。

本阶段目标：
- 为 `multimodal_layer` 增加摘要卡，先展示顶层关键信息，再保留完整 JSON 原文。
- 让多模态层与机器层形成一致的“先摘要、后真值”阅读节奏。
- 不改动后端结构，不新增任何派生接口。

## 原材料来源

本阶段直接参考：
- `/home/admin2/smy/upstream-materials/react-bits-main/src/ts-default/Components/BorderGlow/BorderGlow.tsx`

## 适配原则

- 保留上游 `BorderGlow` 的核心语义：用强调型信息卡提升结构化信息的边界感。
- 不引入指针光效和额外依赖，只复用“高价值摘要卡”的展示语义。
- 摘要卡只从现有 `multimodal_layer` 顶层键中提取类型与简短预览，不发明新的业务字段。
- 完整 JSON 原文继续保留，摘要卡只做入口层压缩。

## 变更范围

### 多模态层
- 在多模态层原始 JSON 前新增摘要卡区域。
- 每张摘要卡至少包含：键名、值类型、简短预览。
- 对对象、数组、字符串采用轻量差异化预览策略。

### 结果页样式
- 复用机器层摘要卡的展示语义，但保持多模态层自己的标题上下文。
- 不改动脚本层、平台层、机器层已完成结构。

## 当前阶段明确不做

- 不改动后端 schema。
- 不隐藏或替换原始 JSON。
- 不新增客户端状态机或动画依赖。

## 验证要求

- 变更后执行 `npm run build`。
- 同步更新 `docs/changelog.md`。
- 以同一批次 Git 提交收口。
