# M81 首次使用闭环收口

## 1. 对应里程碑

- `M-P3-B 首次使用闭环顺滑`

## 2. 背景

此前四页主链已经基本可演示，但首次使用者仍有两个容易卡住的点：

- 首页虽然有强展示，但还不够明确告诉第一次来的用户先做什么
- 结果页虽然信息很重，但还不够明确告诉用户结果出来后下一步做什么

这会让“能演示”和“第一次就能顺利走完”之间仍有距离。

## 3. 本轮实现

- 参考 `react-bits-main` 的 `BorderGlow` 结构语义，新增项目内适配版
- 首页新增 First Run Guide，明确三步新手路径
- 结果页新增 What Next Guide，明确结果后的导出、回看依据与重新开始动作

## 4. 完成效果

这轮之后：

- 首页能明确回答“我现在该先做什么”
- 创建页能明确回答“我怎么开始填”
- 生成页能明确回答“系统现在在做什么”
- 结果页能明确回答“我下一步该做什么”

因此 `M-P3-B` 可以从“未完成”推进到“已完成”。
