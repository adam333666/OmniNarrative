# M85 内部控制台中枢收口

## 1. 对应里程碑

- `M-P4-A 内部工具可直接使用`
- `M-P4-B 内部控制台叙事统一`

## 2. 背景

此前内部页已经各自可用：

- 趋势控制台可查看、可刷新
- checkpoint 控制台可查看、可恢复

但它们仍有一个共同问题：

- 页面之间缺少统一的工具链说明
- 用户需要自己理解“这两页分别负责什么、应该怎么跳转”

这会让内部工具虽然能用，但不够像一组完整控制台。

## 3. 本轮实现

- 新增共享的 `InternalConsoleHub`
- 趋势控制台与 checkpoint 控制台都接入同一套中枢卡片
- 中枢卡片统一解释：
  - 每个工具页负责什么
  - 当前页是哪一页
  - 另一页如何跳转
  - 当前 checkpoint 页对应哪个 generation

## 4. 完成判断

这轮之后：

- 内部趋势页可查看、可刷新
- checkpoint 页可查看、可恢复
- 两张内部页具备统一叙事和统一视觉层级

因此 `M-P4-A` 和 `M-P4-B` 都可以推进为已完成。
