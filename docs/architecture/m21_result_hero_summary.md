# M21 左侧结果总览卡摘要增强

## 目标

在 `M20` 完成导出动作面板增强之后，继续整理结果页左侧顶部的结果总览卡，把它从“标题 + MetaRow 列表”升级成更统一的摘要结构。

本阶段目标：
- 将主题、平台、内容定位、风格摘要重组为更清晰的摘要卡。
- 保留现有请求摘要与结果总览真值，不新增后端接口。
- 让左侧顶部结果总览卡与分析面板、导出动作区形成统一的卡片阅读节奏。

## 原材料来源

- `/home/admin2/smy/upstream-materials/react-bits-main/src/ts-default/Components/ProfileCard/ProfileCard.tsx`

## 适配原则

- 保留上游 `ProfileCard` 的核心语义：通过更强的信息分区来强化主身份信息与关键摘要信息的层次。
- 不引入头像、倾斜交互或其他装饰性行为，只借“主标题 + 摘要卡”的信息组织方式。
- 左侧结果总览卡只重组现有 `request_summary` 与 `result_package.overview` 字段，不创造新的业务字段。
- 主标题与一句话总结仍然作为结果页首焦点，不被摘要卡替代。
