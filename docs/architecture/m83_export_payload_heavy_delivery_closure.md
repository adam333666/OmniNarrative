# M83 导出物重型交付级收口

## 1. 对应里程碑

- `M-P0-C 导出物达到“重型交付级”`

## 2. 背景

在此前阶段中：

- 结果页已经具备较重的执行信息和传播策略信息
- 导出链也已经成立

但两者之间仍有明显落差：

- Markdown 更像摘要文档
- video payload 仍偏下游最小消费载荷

这会导致导出物无法完整体现“页面与导出同源且都很重”的产品卖点。

## 3. 本轮实现

- Markdown 导出新增：
  - 趋势依据与传播增强章节
  - 趋势来源轨迹
  - 标题/钩子/封面备选
  - 评论引导与发布时间建议
  - 角色一致性、场景描述、关键镜头提示、镜头时长建议
- video payload 新增：
  - 场景推进、运动提示、素材道具、视觉参考
  - 发布时间建议、分发角度、缩略图候选、评论引导、封面备选
  - 标题备选、钩子备选、剪辑检查项、CTA 变体
  - 趋势来源轨迹
- 前端导出预览摘要同步变重，直接反映新增导出字段

## 4. 完成判断

这轮完成后，导出物已经不再只是页面的轻量附属品，而是能作为“制作交付级”材料独立展示。

因此 `M-P0-C` 可以从未完成推进到已完成。
