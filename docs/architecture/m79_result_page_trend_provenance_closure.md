# M79 结果页趋势来源轨迹闭环

## 1. 对应里程碑

- `M-P1-C 趋势增强演示闭环`

## 2. 背景

在 `M78` 之后：

- 创建页已经可以看到趋势来源轨迹
- 内部趋势控制台已经可以查看趋势来源轨迹

但主演示链仍有一个明显断点：

- 结果页虽然已经能展示趋势影响和传播策略
- 却还不能直接说明“最终结果参考了哪些趋势来源条目”

这会让演示闭环在最后一段仍需要靠口头解释补齐。

## 3. 本轮实现

- 在结果页分析面板新增趋势来源轨迹展示
- 直接消费结果真值里的 `analysis.trend_summary.source_trace`
- 展示来源名、来源标题、摘录与可跳转外链

## 4. 完成效果

这轮完成后，趋势增强链已经可以在主演示路径里连续看到：

1. 首页说明趋势增强链存在
2. 创建页看到趋势策略与来源轨迹
3. 结果页看到趋势影响与来源轨迹
4. 传播策略区域展示趋势如何改写最终输出

这使 `M-P1-C` 可以从“进行中”推进到“已完成”。
